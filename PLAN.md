# ChemApex · 化学登顶 开发方案

> 参考 app：PhysicsApex（工程目录 `../physicsapex`），本项目是 Apex 家族的化学版，
> 最大化复用其架构与设计语言，但灵魂模块按化学学科特性"换心脏"。

## 一、产品定位

**一句话**：从初中的瓶瓶罐罐，到高考的守恒推断，再到竞赛的轨道之眼——带你看见反应背后的棋局。

- **主战场**：高考化学（高中），资源占 ~80%。
- **入门层**：初中化学，现象与直觉，低门槛起点。
- **天花板**：竞赛/强基点缀（结构化学），"向上仰望"的付费钩子。

**核心判断**：化学不是物理换皮。物理的灵魂是推导，化学的灵魂是
①识局（推断=破案） ②记忆（颜色/现象/方程式） ③巧算（守恒/差量）。

## 二、脊梁：学习主线「登顶之路」（本项目最大差异点）

PhysicsApex 的脊梁是"段位"，ChemApex 在段位之上加一条**清晰的线性学习主线**——
首页 Tab 即主线地图，从山脚到山顶一条路径，每个节点 = 一个知识关卡：

```
节点结构：学（知识点卡）→ 练（关卡题目）→ 战（Boss：守恒之眼战例）
解锁规则：上一节点答对 ≥60% 题目即点亮下一节点
```

## 三、灵魂模块

1. **守恒之眼**（对应降维秒杀）：双解对决，常规解 vs 守恒/差量/极值秒杀，含 timeRatio / keyInsight / plainTalk。
2. **元素星图**（化学杀手级视觉资产）：可交互周期表，考点热度着色，随段位/主线点亮 = 进度可视化。
3. **方程式剧本库**（对应定律宇宙）：方程式 = 条件 + 现象 + 考点 + 陷阱（少量/过量分岔重点标注）。
4. **化学神探**（二期出圈模块）：推断题做成逐条揭线索的卡牌推理游戏。
5. **虚拟实验台**（二期）：气体制备装置拼装 + 现象闪卡。

**武器库**：
| 段位 | 武器 |
|------|------|
| 初中 | 质量守恒、化合价规则、溶解度曲线 |
| 高中 | 电子守恒、电荷守恒、原子守恒、差量法、极值法、十字交叉法、关系式法、平均值法 |
| 竞赛 | 热力学循环（盖斯进阶）、轨道与对称 |

**情感钩子**：化学巨人（门捷列夫/拉瓦锡/居里夫人/侯德榜/屠呦呦/莫瓦桑）+ 毒与火档案馆（化学史故事，篇末钩回考点）。

## 五、诚信红线

- 不臆造"某年某卷某题"出处（沿用 Apex 家族红线）。
- **化学特有**：所有现象/颜色/条件必须准确（学生会直接背进考场）；
  方程式必须配平并标注 少量/过量、浓/稀、点燃/Δ/催化剂 等条件分岔。

## 六、技术方案

- SwiftUI / iOS 16 / Swift 5 / XcodeGen，bundle id `com.chemapex.app`。
- DesignSystem / ThemeColors / SM-2 复习 / 错题本 / Streak 整套复用 PhysicsApex。
- 化学式渲染：Unicode 上下标（H₂O / Fe³⁺），无需 KaTeX，化学天然适合纯文本。
- 周期表：MVP 完整呈现前四周期（1–36 号）+ 高频重元素行（Ag/Sn/I/Ba/Pt/Au/Hg/Pb），
  其余周期不臆造、不占位。

## 七、分期路线

- **一期 MVP（本次）**：4-Tab + 登顶之路 13 关 + 守恒之眼 6 战例 + 元素星图 + 方程式剧本库 ~20 条 + 题库 ~25 题 + 错题本 + 巨人/档案馆。`xcodebuild` 可构建。
- **二期**：化学神探（推断推理游戏）、现象闪卡、题库扩充、SM-2 复习完整版。
- **三期**：虚拟实验台、配平小游戏、付费墙（对标 ¥22 买断）、竞赛内容。
